# Stock Dashboard

개인용 주식 종목 대시보드 프로젝트

이 프로젝트는 다양한 기준으로 주식 종목을 정렬하여 빠르게 확인할 수 있는 웹 대시보드를 만드는 것을 목표로 한다.

단순히 기능 구현뿐 아니라 다음과 같은 목적을 가진다.

- 웹 서비스 전체 아키텍처 이해
- React + Spring Boot 기반 풀스택 경험
- Docker 기반 서비스 운영 경험
- 개인 프로젝트를 통한 기술 스택 확장
---

# Project Goal

이 프로젝트의 핵심 목표는 다음과 같다.

1. 주식 종목 데이터를 수집하고 정렬하여 보여주는 대시보드 구축
2. 다양한 기준으로 종목을 빠르게 탐색할 수 있는 UI 제공
3. 개인용 서비스 수준의 웹 아키텍처 구성
4. Docker 기반 배포 환경 구성

---

# Why This Project

기존 증권 서비스에서도 종목 정렬 기능은 존재하지만 다음과 같은 한계가 있다.

- 특정 기준으로만 정렬 가능
- 사용자 맞춤 기준 설정 어려움
- 빠르게 비교하기 어려운 UI

이 프로젝트는 다음과 같은 목적을 가진다.

- 사용자 정의 기준으로 종목 정렬
- 빠르게 시장 상황 파악
- 개인용 데이터 대시보드 구축
